## 🎯 Ключевые бизнес-правила (Logic)
1. **Формула КРК (Командный Рейтинговый Коэффициент):** 
   КРК = (Базовый рейтинг/оценки * 0.6) + (Коэффициент сплочённости * 0.3) + (Бонус за челленджи * 0.1) [9].
2. **Механика «Спасения»:** Команды могут помогать друг другу, получая за это бонусные баллы [10].
3. **Биржа знаний:** Peer-to-peer доска объявлений для поиска экспертов среди студентов [11].
4. **Анонимное голосование:** В конце цикла участники оценивают вклад друг друга по 5-балльной шкале [10].
