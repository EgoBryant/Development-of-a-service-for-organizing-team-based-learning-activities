# 🏆 Образовательная платформа «Командный зачёт»

Цифровая экосистема для проведения образовательной игры, направленной на снижение стресса при подготовке к сессии и развитие командного взаимодействия в УрФУ.

## 🛠 Технологический стек

* **Frontend:** React / Next.js — для реализации многостраничной архитектуры и роутинга.
* **Backend:** ASP.NET Core (C#) — серверная часть и обработка бизнес-логики.
* **Database:** PostgreSQL — хранение данных пользователей, команд и истории КРК.
* **DevOps:** Docker, Nginx, GitHub Actions (CI/CD).
* **Auth:** OAuth 2.0 / Active Directory УрФУ.

## 🗺 Дорожная карта (Roadmap)

### Фаза 1: Проектирование (30 марта — 12 апреля)
* **Дизайн:** Разработка карты сайта и прототипов страниц ЛК, рейтинга и биржи знаний.
* **Backend:** Проектирование схемы БД и эндпоинтов для авторизации через вузовскую систему.
* **DevOps:** Настройка репозитория и базового окружения для разработки.

### Фаза 2: Core-функционал и логика (13 апреля — 3 мая)
* **Модули 1-2:** Разработка страниц Личного кабинета и функционала управления командой.
* **Модуль 3:** Реализация алгоритма КРК (60/30/10) и страниц глобальных лидербордов.
* **Frontend:** Настройка навигации между основными разделами приложения.

### Фаза 3: Социальные механики (4 мая — 17 мая)
* **Модули 4-5:** Создание страниц «Биржа знаний», «Челленджи» и новостной ленты.
* **Модуль 6:** Реализация форм еженедельных отчетов и механизма «Спасения».
* **Интеграция:** Настройка импорта данных из учебного портала для расчета базового рейтинга.

### Фаза 4: Тестирование и запуск (18 мая — 1 июня)
* **QA:** Проверка корректности переходов, нагрузочное тестирование и аудит безопасности.
* **Admin:** Разработка панели организатора для модерации и управления событиями.
* **Релиз:** Финальный деплой и проведение пилотного запуска на потоке.

## 📊 Технические требования
* **Производительность:** Время отклика интерфейса при переходах не более 2 секунд.
* **Надежность:** Время безотказной работы системы не менее 99%.
* **Безопасность:** Ролевая модель доступа и защита персональных данных.
